# CPlotter
 Программа для рисования графиков в консоли и текстовых файлах
 
 Пример работы:
```
                   Hello, world! I'm a graph
****************************************************************
*                               |                              *
*                xx             |    x                  xx     *
*               x  x            |  xx xx               x  x    *
*x             x    x           | x                   x    x   *
*             x      x          |       x            x      x  *
* x          x                  |x       x          x          *
*--x------------------x---------x---------x------------------x-*
*           x          x       x|                  x          x*
*   x      x            x       |          x      x            *
*    x    x                   x |           x    x             *
*     x  x               xx xx  |            x  x              *
*      xx                  x    |             xx               *
****************************************************************
```
