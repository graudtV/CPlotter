void plotSetFunction ( Plot *  plot,
double(*)(double)  func

Настраивает функцию, график которой нужно построить

Parameters
double(*func)(double) - Указатель на функцию Пример использования:
#include <math.h>
Plot plot = {};
plotSetFunction(&plot, cos); //Функция cos определена в math.h
... // Инициализация остальных значений
plotDraw(&plot);
plotDestroy(&plot);
Чтобы нарисовать свою функцию, нужно определить её и точно также передать указатель на неё. Пример:
double someFunc(double x)
{
return 1 / x;
}
plotSetFunction(&plot, someFunc);
